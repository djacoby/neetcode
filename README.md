# NeetCode - Blind 75

Solutions to the [Blind 75](https://neetcode.io/) problems written in TypeScript.

## Setup

```sh
pnpm install
```

## Running Solutions

```sh
pnpm solve src/<problem-folder>/<file>.ts
```

For example:

```sh
pnpm solve src/01-arrays-and-hashing/contains-duplicate.ts
```
